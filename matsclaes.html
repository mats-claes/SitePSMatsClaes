Orientation Project 1 (groupwork)

Let's make a LAN party!

In groups of 5, we had to set up a server that could host a particular game. We chose the game Minecraft. We received a switch and a server with no configuration on it. We had to configure everything ourselves: from installing the game to connecting to the Internet. Finally, the intention was that the other groups could play through our switch on our server.

What we learned here was how to configure an empty server. It was our first time working with physical devices. Before we only got the theory and now it is finally up to us to do something with it. It was very instructive to be able to work on a project with other students for a week and to push each other to the limit for a nice end result.

Second year projects

Project Unicenta (groupwork)
Systems Engineering & Analysis

For the Systems Engineering & Analysis course, we were divided into groups of 3. In this group we had to create a solution at the request of a fictitious customer. The question was that supermarkets had a way to see which products were going to expire in a short period of time and then these products could be picked up by a Non-Profit Association.

For this we devised a system that would solve this efficiently. Using RFID and a Raspberry Pi we were able to realize this project. Each product was equipped with an RFID chip with the expiration date in it. When the product arrives in the store, this product will be entered into the database. When the product is purchased, the product will be removed from the database. But the products that were approaching the expiration date were entered into a program and the stock manager could see which products were which and when they would expire and the staff could take the products off the shelves. That way the associations could be notified so they could pick up the products.

From this I learned that it is best to start well in advance and have a good plan in mind. Throughout the week we changed approaches a few times. You can already save a lot of time from this.

Project Unicenta

Project Hosting

Project Hosting is a subject where you have to work on a project in group all year. The project involves us setting up a hosting platform. It does not matter what techniques and infrastructure we have to use. The goal is that students of 2APP/AI can host their PHP projects on our infrastructure. Throughout the year, we will be meeting every Thursday to brainstorm how to bring this to a successful conclusion. All the learning we have seen now and in the past year we now need to take together. Quite a job that involves a lot of work and stress. 

In the last week of class this project should be finished. For the evaluation we have to come to a jury and give a demonstration of how we have achieved it and how users can use our infrastructure.  

What I have already learned is that there is a lot of work to be done at home. The time in class is much too short. A lot of energy and perseverance is very necessary for this subject.
